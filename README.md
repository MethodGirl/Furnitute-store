## 🛋️ Furniture Store

**Furniture Store** — проект интернет-магазина мебели.

## Описание проекта

Многостраничный сайт, имитирующий интерфейс интернет-магазина мебели: каталог товаров, фильтры и корзину.
Проект создан для практики многостраничной верстки и интерактивности интерфейса с использованием JavaScript.

![Preview](project.gif)

## Технологии

- HTML5
- CSS3
- SCSS
- JavaScript (Vanilla JS)

## Основной функционал

- **Модальное окно корзины** — при наведении на иконку корзины появляется окно с товарами. Можно открыть корзину и оформить заказ.
- **Фильтрация товаров** — интерфейс фильтров реализован через `checkbox` и `radio buttons` и подготовлен для дальнейшей логики фильтрации.
- **Интерактивность** — эффекты наведения на элементы интерфейса и работа кнопок.
- **Многостраничная структура сайта** — отдельные страницы для главной, каталога и корзины.
